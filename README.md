# Next.js Portfolio Website

A modern, animated portfolio built with Next.js, React, TypeScript, and Tailwind CSS. Features smooth GSAP-powered animations and a clean, responsive layout.

## Quick Start

**Requirements:** Node.js v18+, npm or pnpm

```bash
npm install --legacy-peer-deps
npm run dev
```

Visit `http://localhost:3000`.

## Scripts

* `npm run dev` – Development server
* `npm run build` – Production build
* `npm start` – Run production server
* `npm run lint` – ESLint checks

## Features

* Hero banner with CTA and resume download
* About, Skills, Projects, and Footer sections
* GSAP scroll-based animations and page transitions
* Custom cursor, particle background, preloader, and scroll progress indicator

## Tech Stack

* Next.js 15.5
* React 19 (RC)
* TypeScript
* Tailwind CSS
* GSAP
* Lucide React

## Notes

* Uses `--legacy-peer-deps` due to React 19 RC peer dependencies
* Resume served from `/public/Resume.pdf`
